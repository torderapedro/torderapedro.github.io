
Este paquete contiene una maqueta minimalista para un portfolio de arquitectura
pensado para publicar en GitHub Pages.

Estructura:
- index.html, index-en.html  -> páginas de inicio (ES / EN)
- cv.html, cv-en.html        -> CV (con enlace a assets/cv.pdf)
- portfolio.html, portfolio-en.html
- blog.html, blog-en.html
- proyectos/                 -> páginas de proyectos (proyecto1.html + versión EN)
- blog/                      -> páginas de entradas de blog (entrada1.html + EN)
- style.css                  -> estilos globales
- assets/                    -> imágenes y PDFs de ejemplo

Sustituye los textos y las imágenes por tu contenido y sube la carpeta al repositorio
llamado tuusuario.github.io para publicarlo con GitHub Pages.
